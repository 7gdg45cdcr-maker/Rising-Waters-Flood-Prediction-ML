# Rising Waters - Flood Prediction

## Project Overview

This project predicts the possibility of floods using Machine Learning based on environmental parameters.

### Features

- Rainfall input
- Water Level input
- Humidity input
- Logistic Regression model
- Web interface using Flask

## How to Run

Install dependencies:

```
pip install -r requirements.txt
```

Run:

```
python app.py
```

Open browser:

```
http://127.0.0.1:5000
```
